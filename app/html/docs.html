KD:M API - A RESTful API for Monster, by Kingdom Death

The KDM API at https://api.kdm-manager.com is a RESTful API for campaign/settlement management in "Monster" by Kingdom Death.

The API is a fan-created and fan-maintained project that is not supported by, affiliated with or otherwise authorized by Kingdom Death.

Some parts of the API are free to use. Others require an API key. Endpoints that require an API key to access are indicated below by the key (🔑) icon.
