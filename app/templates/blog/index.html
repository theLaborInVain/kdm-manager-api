OS and browser support

In order to support the largest number of users, The Labor in Vain publishes and provisions the KD:M API and its related web applications as an OS- and hardware- agnostic, browser-accessible, open software platform.

The platform is optimized for the OS and browser combination used most frequently by users, which is currently:

  • OS: Windows 10
  • Browser: Microsoft Edge

The platform will never be published or provisioned as an OS- or platform-specific software package, e.g. iOS/Android native application.
